## 可用指令列表

### 1. 飞行模式切换
`/cfly toggle`
- **功能**: 开启/关闭创造模式飞行

### 2. 鞘翅加速控制 
`/cfly elytratoggle`
- **功能**: 开关鞘翅飞行增强系统
- **默认状态**: 开启
- **智能联动**:
  - 开启时自动适配Tweakeroo飞行参数
  - 禁用时恢复原版鞘翅物理

### 3. 速度调节系统
`/cfly speed <数值>`
- **参数范围**: 任意非负数 (建议 0.1-5.0)
